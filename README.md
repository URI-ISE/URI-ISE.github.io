URI-ISE.github.io

Static website for the University of Rhode Island Industrial & Systems Engineering group. The root page introduces the lab mission and research areas and links to project demos.

Pages
- /: Lab overview, mission, research areas, and contact info.
- /hanoi/: Interactive Tower of Hanoi solver and visualizer with step controls.
- /rsvp-reading/: RSVP Reader web app for speed reading (PDF/EPUB support), configured as a PWA.

Tech
- Plain HTML/CSS/JS.
- RSVP Reader assets are prebuilt and served from rsvp-reading/assets.
- Service worker + manifest enable offline-capable PWA behavior.

Deploy
- Hosted via GitHub Pages from the main branch.
